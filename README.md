# 安装说明

安装环境：win10，win11 64位操作系统
安装路径要求
请勿将本软件安装到以下目录：
C:\Program Files\
C:\Program Files (x86)\
任何系统受保护的文件夹
推荐安装路径：
D:\SmartVillageSystem\
E:\SmartVillageSystem\
F:\SmartVillageSystem\
如果安装程序未自动建议上述路径，请手动修改为合适的目录。
关于杀软提示
本软件未购买微软数字签名证书，安装或运行时部分杀毒软件可能会弹出拦截提示。
这是未签名程序的正常现象，并非软件存在安全问题。
建议： 安装过程中如遇到杀软拦截，请临时关闭杀毒软件后重新安装。
安装完成后可恢复开启。
本软件不含任何恶意代码。如对安全性有疑虑，可选择不安装使用。
